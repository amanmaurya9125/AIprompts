#  AI Prompts App

An Android application that provides a curated collection of AI prompts to help users generate better responses from AI tools. The app is built using **modern Android development practices** with a clean UI and offline support.

##  About the App

* Browse high-quality AI prompts
* Copy prompts instantly
* Save favorite prompts for later use

The app is designed with a user-friendly interface using Jetpack Compose.

##  Features

*  Category-wise AI prompts
*  Save prompts to Favorites
*  One-tap copy to clipboard
*  Offline support using local database
*  Clean & modern UI
*  
##  Tech Stack

* Language: Kotlin
* UI: Jetpack Compose (Material 3)
* Architecture: MVVM (Model-View-ViewModel)
* Database: Room Database
* Data Storage:
         Room (Favorites)
         SharedPreferences (settings & flags)
* Lists: LazyColumn
* Data Format: JSO
